# 📚 Library Manager

A simple library management application built with React and Node.js.

## Features

- Book management
- Author management
- Member management
- Loan management
- Search books

## Technologies

### Frontend

- React
- JavaScript
- Vite
- CSS

### Backend

- Node.js
- Express
- PostgreSQL
- Prisma

## Project Status

🚧 In development.

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm run dev
```
